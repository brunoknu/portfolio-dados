# Análise de Vendas — Power BI

## Objetivo
Analisar dados de vendas de um e-commerce fictício para identificar faturamento,
ticket médio, produtos mais vendidos e performance mensal.

## KPIs
- Faturamento total
- Ticket médio
- Quantidade de vendas
- Top produtos

## Ferramentas
- Power BI
- SQL
- Excel

## Principais Insights
- Concentração de faturamento em poucos produtos
- Crescimento sazonal em determinados períodos
